项目简介

HZQ-WIKI 是一款针对IT团队开发的简单好用的在线文档管理系统。

HZQ-WIKI 是基于 Vue + Java + Mysql 开发的一款文档管理系统。

可以用来储存日常接口文档，数据库字典，手册说明等文档。内置文档空间管理，文档管理,用户管理，权限管理等功能，能够满足大部分中小团队的文档管理需求。

主要功能

  1. 文档空间管理：可以对空间进行编辑更改，成员添加等。
  2. 文档管理：添加和删除文档，成员添加等。
  3. 文档空间加密：可以设置项目公开状态，私有项目需要通过Token访问。
  4. 文档加密：可以设置项目公开状态，私有项目需要通过Token访问。
  5. 用户管理：管理员(admin)添加和修改用户等。
  6. 个人中心：对个人基础信息进行修改等。
  7. 密码修改：用户自己修改密码。
  8. 配置管理：根据客户需求，自定义一些站点信息
